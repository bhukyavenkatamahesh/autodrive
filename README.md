# AutoDrive

Starter monorepo scaffold for a vehicle marketplace platform with:

- `frontend/` for the Next.js 14 web app
- `services/` for backend and ML microservices
- `infra/terraform/` for Azure infrastructure as code
- `infra/helm/` for Kubernetes deployment templates
- `.github/workflows/` for CI/CD automation

Use `docker compose up --build` or `make dev` to start fleshing out local development.
